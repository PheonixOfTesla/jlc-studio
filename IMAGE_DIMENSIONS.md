## Recommended Dimensions

### Hero Sections
- **Desktop**: 1920 x 1080 (16:9) or 1920 x 800 (wide banner)
- **Mobile**: 750 x 1334 (portrait) or responsive crop

### Featured/Gallery Grid
- **Large items**: 1200 x 1600 (3:4 portrait)
- **Standard items**: 800 x 800 (1:1 square) or 800 x 1000 (4:5)

### Profile Images
- **Square format**: 1024 x 1024 (1:1)

### Logo
- **Current**: 1024 x 1024 (can be scaled down as needed)
- **Navbar usage**: displays at 50px height
- **Footer usage**: displays at 60px height
